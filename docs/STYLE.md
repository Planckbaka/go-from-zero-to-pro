# Style Guide (Repo)

## Go code
- Run `gofmt` on all Go files.
- Follow Go Code Review Comments:
  <https://go.dev/wiki/CodeReviewComments>
- Additional reference: Google Go Style Guide
  <https://google.github.io/styleguide/go/>

## Docs (Markdown)
- Prefer small code snippets and link to full runnable examples under `examples/`.
- Each chapter should include:
  - Goals
  - Minimal example
  - Pitfalls
  - Exercises (optional)
